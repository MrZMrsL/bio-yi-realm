> **状态**: 设计阶段 — 所有数值标记为 `[PLACEHOLDER]`，待原型验证后定稿

## 🎯 设计支柱（Design Pillars）

1. **知识即力量** — 你的学科知识直接转化为战斗伤害，答对越多越强
2. **专精有意义** — 选择专精不只是"出什么题"，它改变你的游戏风格、成长路径和策略选择
3. **涌现式策略** — 装备、宠物、符文、专精四者交叉产生非预设的强力组合
4. **渐进式挑战** — 难度随玩家成长平滑上升，始终在"舒适区边缘"施加压力
5. **回馈循环** — 每次游戏操作都产生有意义的反馈（经验、材料、图鉴、成就）

## 📊 现状分析：专精系统当前行为

目前的专精筛选逻辑（`getQuestionsForFloor()`）已经正确实现了：

| 专精 | 筛选行为 |
|------|---------|
| `chem` | 只出 `subject === 'chem'` 的题 |
| `bio` | 只出 `subject === 'bio'` 的题 |
| `yi` | 只出 `subject === 'yi'` 的题 |
| `biochem` | 50%概率 `chem` / 50%概率 `bio` |
| `all` | 不筛选，所有学科混出 |

**但仅此而已。** 这意味着：

- ❌ 选化学专精的玩家和选全部专精的玩家，战斗体验完全一样（只是题不同）
- ❌ 专精对玩家属性、成长、策略选择没有影响
- ❌ 没有专精专属的玩法内容
- ❌ 没有鼓励玩家尝试不同专精的动机

## 🔧 核心改动：专精系统深化

### 1.1 专精属性加成

每个专精在游戏开始时提供**独特的初始属性加成**和**成长倾向**：

| 专精 | 初始加成 | 升级倾向 | 设计意图 |
|------|---------|---------|---------|
| 🧪 **化学专精** | 攻击+5，暴击率+5% | 每3级额外+2攻击 | 玻璃大炮，高伤害高风险 |
| 🧬 **生物专精** | 生命+30，防御+3 | 每3级额外+15生命 | 持久战，容错率高 |
| ☯️ **易学专精** | 闪避率+5%，经验+10% | 每3级额外+1属性点 | 成长型，后期强 |
| 🔬 **生化专精** | 攻击+2，生命+15 | 每3级随机获得1项化学或生物加成 | 均衡型，适应性广 |
| 🌟 **全部专精** | 全属性+2，经验+5% | 每3级全属性+1 | 后期全面型，前期较弱 |

> **实现方式**：在 `setSpecialization(spec)` 时应用初始加成，在 `winBattle()` 的升级逻辑中增加专精成长分支。

### 1.2 专精技能树

每个专精拥有独立的**3个被动技能**，随等级解锁：

```
化学专精技能树：
  Lv.5  ─ 燃烧反应：答对化学题时，额外造成50%持续伤害（3回合）
  Lv.15 ─ 催化增效：使用药水时效果翻倍
  Lv.30 ─ 元素爆破：化学题连击上限+2（最高5连）

生物专精技能树：
  Lv.5  ─ 细胞再生：每场战斗开始时恢复20%生命
  Lv.15 ─ 基因适应：答错时只受50%反噬伤害
  Lv.30 ─ 共生体：宠物提供的属性加成翻倍

易学专精技能树：
  Lv.5  ─ 卦象预判：战斗开始时有30%概率闪避敌人第一次攻击
  Lv.15 ─ 阴阳调和：每答对3题，恢复10%生命
  Lv.30 ─ 太极领域：每场战斗有1次机会将答错的题"重roll"一次

生化专精技能树：
  Lv.5  ─ 交叉反应：连续答对不同学科题目时，连击伤害额外+25%
  Lv.15 ─ 双重打击：答对题时20%概率触发双倍伤害
  Lv.30 ─ 学科融会：化学题和生物题共享连击计数

全部专精技能树：
  Lv.5  ─ 博闻强识：所有学科题目都有10%概率触发额外经验+5
  Lv.15 ─ 融会贯通：每答对一个不同学科的题，攻击永久+1（上限20）
  Lv.30 ─ 万象归宗：战斗中可消耗一次机会切换当前出题学科
```

## 🔄 核心游戏循环（深化版）

### Moment-to-Moment（0-30秒）
- **择专精而入** → 选方向定风格
- **地牢探索** → 选房间、看预览、做抉择
- **答题战斗** → 读题、思考、选择、看结果
- **即时反馈** → 正确→伤害+连击+音效｜错误→反噬+错题入库

### 地牢循环（5-30分钟）
- **战前准备** → 装备/宠物/药水配置
- **房间探索** → 9间密室，含1间Boss房
- **战斗** → 知识对决，连击暴击，掉落加成
- **清场或速通** → 全清→额外奖励｜跳过→时间效率
- **层数推进** → 难度递增，元素轮换

### 长期循环（数小时-数周）
- **等级成长** → 属性点、称号解锁、技能树激活
- **装备锻造** → 收集材料→打造神器→提升战力
- **宠物养成** → 捕捉→升级→携带战斗→元素克制
- **图鉴收集** → 怪物/材料/鱼/古籍/符文全收集
- **成就殿堂** → 里程碑+挑战性成就
- **周常Boss** → 每周限时挑战，限定奖励
- **限时挑战赛** → 见下文新增玩法

## 🆕 新增玩法模块

### ⚡ 模块一：学识之塔（Endless Tower）

**定位**：独立于地牢的纯答题挑战模式，验证真实知识水平

**设计**：
- 独立于地牢的无限爬塔模式，入口在"地牢探索"旁的独立卡片
- **不消耗角色生命**，纯答题模式
- 每层1道题，答对上1层，答错停留
- 连续答对获得**学识积分**（独立货币）
- 每10层一个Boss题（高难度，且必须答对才能继续）
- 问题学科**完全由专精决定**（选化学→纯化学题）

**学识积分兑换**：
| 物品 | 积分 | 限量 |
|------|------|------|
| 经验药水（+50经验） | 100 | 每日3次 |
| 随机装备箱 | 300 | 每日1次 |
| 限定称号「博学之士」 | 1000 | 1次 |
| 全学科重置卷 | 500 | — |

**平衡要点**：
- 学识积分不与地牢经验挂钩，避免"代练"式不平衡
- 每日上限 `[PLACEHOLDER]` 次重置机会

### ⚡ 模块二：元素秘境（Elemental Rift）

**定位**：每周轮换的限时属性主题副本，考验元素策略

**设计**：
- 每周开放一个**元素秘境**（水/火/酸/电/冰/风轮换）
- 秘境中**只有当前元素的敌人**出现
- 携带**克制该元素的宠物**时，获得 `[PLACEHOLDER]` 倍伤害加成
- 秘境共5层，全清后获得**该元素的专属材料**（锻造高级装备用）
- 秘境Boss有**元素护盾**：只有答对**对应学科**的题才能破盾

**秘境轮换表**：
| 周次 | 元素 | 克制宠物 | 限定掉落 |
|------|------|---------|---------|
| 第1周 | 🔥 火 | 水/冰宠物 | 火焰核心×3 |
| 第2周 | 💧 水 | 酸/电宠物 | 水之精华×3 |
| 第3周 | 🧪 酸 | 水/电宠物 | 酸液结晶×3 |
| ... | ... | ... | ... |

### ⚡ 模块三：符文系统（Rune System）

**定位**：装备之外的独立成长维度，提供定制化战斗风格

**设计**：
- 玩家有6个符文槽（3个主动，3个被动）
- 符文通过**钓鱼**、**周常Boss**、**学识之塔**获得
- 符文分3个稀有度：普通（白）/ 稀有（蓝）/ 传说（金）

**符文示例**：

| 名称 | 类型 | 稀有度 | 效果 |
|------|------|--------|------|
| 答题加速 | 被动 | 普通 | 每场战斗第一次答错不反噬 |
| 金币嗅觉 | 被动 | 稀有 | 战斗金币掉落+30% |
| 连击护盾 | 被动 | 传说 | 连击中断时保留上一半连击数 |
| 学科转换 | 主动 | 稀有 | 消耗一次行动，下一题指定学科出现 |
| 紧急回避 | 主动 | 普通 | 战斗中可强制逃跑一次，不损失资源 |
| 时间裂隙 | 主动 | 传说 | 限时Boss战中，暂停倒计时5秒 |

**平衡要点**：
- 符文槽位通过**等级**解锁（Lv.1/10/20/30/40/50各开1个）
- 主动符文冷却时间 `[PLACEHOLDER]` 场战斗
- 同一传说级符文不可叠加

### ⚡ 模块四：限时挑战赛（Daily/Weekly Challenge）

**定位**：每日/每周刷新固定参数挑战，比拼答题实力

**设计**：

**每日挑战**：
- **固定种子**：所有玩家遇到相同的题目序列（公平竞技）
- **限制条件**：如"只能用防御装备"、"不允许使用药水"
- **评分方式**：答对率 + 耗时（秒）→ 综合评分S/A/B/C/D
- **奖励**：评分越高奖励越好（金币+学识积分）

**每周挑战**：
- **专精周赛**：本周只能使用某个专精进入
- **排行榜**：按最高层数排名（仅本地，或可选分享）
- **奖励**：限定称号、传说符文碎片

**设计意图**：
- 每日挑战为玩家提供**每日上线理由**
- 每周挑战鼓励玩家**尝试不同专精**
- 排行榜提供**社会比较驱动**

---

### ⚡ 模块五：专精深潜副本（Mastery Dungeon）

**定位**：专精专属的挑战副本，只有对应专精玩家可进入

**设计**：
- 每个专精有独立的**深潜副本**（共5个）
- 副本共15层，每层只有该专精学科题目
- 第5/10/15层有**专精守护者Boss**，拥有专精特化能力
- 通关后获得**专精限定装备**（只有该专精玩家可装备）

**专精限定装备**：

| 专精 | 装备名称 | 类型 | 效果 |
|------|---------|------|------|
| 🧪 化学 | 元素调和之杖 | 武器 | 答对化学题时 + `[PLACEHOLDER]` 额外伤害 |
| 🧬 生物 | 基因螺旋护甲 | 防具 | 每场战斗回复 `[PLACEHOLDER]` %生命 |
| ☯️ 易学 | 太极玄光佩 | 饰品 | 闪避率 + `[PLACEHOLDER]` % |
| 🔬 生化 | 双螺旋指环 | 饰品 | 生化专精技能触发率 + `[PLACEHOLDER]` % |
| 🌟 全部 | 万象归一之冠 | 防具 | 全属性 + `[PLACEHOLDER]`（全部专精特有） |

**设计意图**：
- 给玩家选择专精的**成就感**和**专属奖励**
- 鼓励多周目体验不同专精
- 限定装备不可跨专精使用，保持专精特色

## ⚖️ 平衡性设计

### 难度曲线

```
楼层   | 难度 | 敌人属性倍率 | 题目难度分布
1-5    | 入门 | ×1.0-1.2     | 80%简单 + 20%中等
6-10   | 普通 | ×1.2-1.6     | 50%简单 + 50%中等
11-15  | 中等 | ×1.6-2.2     | 30%简单 + 60%中等 + 10%困难
16-20  | 困难 | ×2.2-3.0     | 50%中等 + 50%困难
21-30  | 噩梦 | ×3.0-4.5     | 30%中等 + 70%困难
30+    | 地狱 | ×4.5+        | 100%困难 + 实验题概率出现
```

### 专精难度映射

| 专精 | 1-10层 | 11-20层 | 21-30层 | 30+层 |
|------|--------|---------|---------|-------|
| 化学 | 化学题量充足，流畅 | 中等 | 开始吃力 | 高难度 |
| 生物 | 生物题量充足，流畅 | 中等 | 开始吃力 | 高难度 |
| 易学 | ⚠️ 目前易学题较少 | 明显不足 | 溢出挑战 | 极难 |
| 生化 | 双学科池，题量充足 | 流畅 | 中等 | 困难 |
| 全部 | 三学科池，题量最大 | 流畅 | 流畅 | 考验全面性 |

> 🔴 **注意**：当前 `yi_medium.js` 仅1个文件，化学有6个文件，生物有5个文件。**易学题库严重不足**，建议补充至至少与化学/生物同等量级。
