# 生化易界冒烟测试（发布前必做）

> ⚠️ **铁律**：每次 push 到 master 之前，必须逐项勾选。漏一项 = 带病发布。

## 核心路径（每次更新必测，5分钟）

- [ ] **启动游戏**：打开 https://mrzmrsl.github.io/bio-yi-realm，确认无白屏/报错
- [ ] **地牢入口**：确认主界面显示正常，"地牢探索"按钮可点击
- [ ] **进入地牢**：点击进入 → 确认显示"第 X 层地牢"入口界面
- [ ] **战前准备**：点击"进入地牢" → 确认显示战前准备（怪物情报、元素克制）
- [ ] **房间选择**：确认进入 → 确认显示 9 个房间格子
- [ ] **普通战斗**：点击任一房间 → 确认进入战斗（题目+选项+血条）
- [ ] **战斗答题**：答对一题 → 确认敌人扣血、连击计数正常
- [ ] **战斗胜利**：答对至敌人 HP 归零 → 确认显示"战斗胜利"+掉落/捕捉提示
- [ ] **领取掉落**：点击领取 → 确认进背包/材料库
- [ ] **进入下一层**：确认全部清空后 → 点击"进入下一层" → 确认层数+1
- [ ] **退出地牢**：点击 X 关闭面板 → 确认回到主界面，无残留状态

## 状态机互斥（v8.0+ 每次更新必测）

- [ ] **战斗中禁止切面板**：战斗中点击底部导航栏（商店/钓鱼/仓库/农场等）→ 确认弹出提示"战斗中不能切换"，页面不跳转
- [ ] **限时Boss独立显示**：点击"限时Boss" → 确认面板内**不显示**"🏰 第 X 层地牢"入口界面
- [ ] **限时Boss战斗**：确认显示限时Boss专属界面（倒计时条、Boss技能）
- [ ] **限时Boss逃跑**：战斗中点击逃跑 → 确认直接回到主界面，无地牢保存提示
- [ ] **关闭面板回主界面**：任何面板点击 X → 确认回到主界面，底部导航正常

## 钓鱼系统（每次更新必测）

- [ ] **打开钓鱼**：点击"垂钓" → 确认显示钓鱼界面
- [ ] **正常钓鱼**：点击"开始钓鱼" → 等待咬钩 → 点击收杆 → 确认钓到鱼/古籍
- [ ] **周期性答题**：连续钓 5 次 → 第 6 次确认触发答题解锁（显示题目+选项）
- [ ] **答对继续**：答对解锁题 → 确认自动开始钓鱼，计数重置为"本轮剩余 5 次"
- [ ] **答错锁定**：答错解锁题 → 确认显示"今日无法继续"，钓鱼按钮禁用
- [ ] **钓到古籍**：钓到古籍 → 点击"研读" → 确认显示题目 → 答题后确认状态切换正常（成功/失败）
- [ ] **研读后不卡住**：研读答题后 → 确认按钮恢复正常，可以继续钓鱼或关闭面板
- [ ] **吃鱼/放生**：钓到普通鱼 → 确认"吃"和"放生"按钮正常，点击后状态回 idle
- [ ] **钓鱼关闭面板**：钓鱼中点击 X → 确认回到主界面，无残留状态

## 周边系统（大版本/功能更新时测）

- [ ] **商店买卖**：打开商店 → 购买物品 → 确认金币扣除、物品进背包
- [ ] **仓库管理**：打开仓库 → 确认装备/材料/消耗品正常显示
- [ ] **农场操作**：打开农场 → 确认宠物列表正常 → 设置跟随/取消跟随
- [ ] **图鉴更新**：新获得怪物/材料/鱼/古籍 → 打开图鉴确认已记录
- [ ] **错题本**：答错题目后 → 打开自习室 → 确认错题已记录
- [ ] **设置存档**：点击"存档管理" → 确认导出/导入/删除功能正常

## 边界情况（重构/大版本时必测）

- [ ] **旧存档兼容**：用旧版本存档启动 → 确认无报错，gameMode 自动推断正确
- [ ] **清空存档重开**：删除存档 → 刷新页面 → 确认新手引导正常触发
- [ ] **背包满拾取**：装备满10件后战斗胜利 → 确认提示"背包已满"，不覆盖旧装备
- [ ] **农场满捕捉**：农场满5只后触发捕捉 → 确认提示"农场已满"
- [ ] **断网/刷新**：战斗中刷新页面 → 确认存档恢复，不丢失进度（或回到合理状态）
